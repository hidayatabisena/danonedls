Introducing: DanoneDLS

Hidayat Abisena

DanoneDLS is a DANIS very own Design Language System that contains collection of brand identity, guidelines and components which can be used to create User Interface and User Experience Design across all of Danone's internal Apps.

In order to bring consistency across all products, a design language systems is necessary. This would ideally give anyone who uses it a comprehensive idea about Danone design philosophy.
